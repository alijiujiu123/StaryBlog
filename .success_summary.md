## 🛠️ 使用指南

### 添加新文章
1. 在 `posts/` 目录创建新的 HTML 文件
2. 在首页 `index.html` 的文章列表中添加链接
3. 运行 `./update.sh` 快速部署

### 快速更新
```bash
# 方式一:使用更新脚本(推荐)
./update.sh

# 方式二:手动更新
git add .
git commit -m "Your message"
git push
ssh root@43.167.189.165 "cd /var/www/staryblog && git pull"
```

### 完整部署
```bash
./deploy.sh
```
